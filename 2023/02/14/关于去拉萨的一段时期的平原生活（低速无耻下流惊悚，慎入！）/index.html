# 好久没写，不知道怎么开始了。

我的确是不具备爱人的能力的，我想。

我有着一种极其倒错的自恋，但这不重要。

重要的是我突然发现这么多年我好像毫无长进，我指的是，很多年前我会病态地不断怀疑所有跟我距离近的人是否背叛了我，现在还是差不多，只不过这个对象从跟我距离近的人变成了我喜欢的人。

之前看晚安布布，印象最深的情节是翠追舅舅到路上，说 “我会负责任把小野寺先生杀死的，所以请不要再逃走了”。为什么会是这段呢，不知道，可能我想要个保障，也可能只是想被好女人杀掉。

我是知道的，这种关系本身就是 “把一个能毁掉自己的按钮交给对方，然后祈求对方不会按下去”，但是知道又有什么用呢，人总是会对不确定和无意义的事物感到恐惧，难道要享受这种恐惧吗？那难度系数未免有点太高了。

随着我在现实生活中逐渐变得 “正常”，我在网路上的表现便好像要把那个 “正常” 的空洞填上一样开始越来越 psychic，准确的说，并不是网路上，而是私人时间，就在近一年多，我开始看了很多之前一点不感兴趣甚至觉得恶心的题材，而也随着我的这些 xp 发展的越来越奇形怪状，将它们表达出去也变得越来越难。这样，一个从前和现在的自己的鲜明对照就产生了：以前的我是完全自闭的，壳子在外面；而现在壳子内化了。

# 用一个词来总结我的性格就是：拧巴

这也不重要。

# 重要的是，他妈的，太多的人，没能清楚认识到什么行为是在表达自己的 fetish，而什么不是，我认识的很清楚，所以我不表达！

这是克己的诅咒，我对异性有种恐惧心理，想不起来因为什么了但就是有，不是所谓 “恐女”，而是不会说话，说不出话。

我的生涯既是打胶。大概是在去年或者前年，我看了很多母系的作品，现在也在看，音声非常不错，有很多全肯定系，听了顿觉自己收到了洗礼，耳目聪明神清气爽。而一想到可能有人会看到这篇东西，我便不能再继续说了，只可至此不可越过。

13 岁之后的绝大多数夜晚我都在寻找打胶素材中度过，或许是被淫秽色情毒害太深才导致了我现在这扭曲的性格，不过要是没有这些东西我早就似了，从这里来说，我必须对我的牛子说声谢谢。

跑到哪里去？不知道。就算是跑到沟里我也不得不跑。

低速, 无耻
